# Valentine's Day Web App

A fun, interactive Valentine's Day web application built with React and Vite.

## Features

- **Interactive Yes/No Buttons**: Click "Yes" to answer the Valentine's question
- **Dodging No Button**: The "No" button moves around the screen when you try to hover over it
- **Thank You Page**: After clicking "Yes", you'll be taken to a beautiful thank you page with animations
- **Responsive Design**: Works great on desktop and mobile devices
- **Smooth Animations**: Beautiful transitions and animations throughout

## Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to the URL shown in the terminal (usually `http://localhost:5173`)

## Build

To create a production build:
```bash
npm run build
```

The built files will be in the `dist` folder.

## Technologies Used

- React 18
- Vite
- CSS3 Animations
